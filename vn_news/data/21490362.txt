Hướng dẫn chuyển file Word sang PDF trên Office 2016
Microsoft Office 2016 là bộ công cụ hỗ trợ người dùng trong việc soạn thảo văn bản, tạo bảng tính, tạo video trình chiếu… Bên cạnh đó, nó cũng cho phép chuyển đổi file Word sang PDF. Bài viết sau đây sẽ hướng dẫn bạn thực hiện thao tác này.
Hiện nay, Microsoft Word là một trong những định dạng văn bản phổ biến nhất trên thế giới. Tuy nhiên, Word không phải là định dạng được đánh giá cao trong vấn đề bảo mật so với PDF. Vì vậy, cách tốt nhất trước khi bạn muốn upload một tài liệu Word nào đó lên mạng internet, trước hết hãy chuyển đổi chúng sang định dạng PDF. Hiện nay, cũng có rất nhiều phần mềm, tiện ích, website giúp đỡ bạn chuyển Word sang PDF dễ dàng. Bài viết này sẽ hướng dẫn bạn chuyển file Word sang PDF trên Office 2016. Bước 1: Mở file Word bạn cần chuyển đổi. Sau đó, nhấp chọn thẻ TỆP. Bước 2: Chọn Lưu Như. Tiếp đến, bấm chọn Duyệt, để chọn vị trí lưu file PDF. Bước 3: Chọn nơi lưu file, trong mục Save as type các bạn chọn PDF. Bước 4: Xuất hiện một số tùy chọn cho file PDF bạn muốn lưu như: - Chuẩn: lựa chọn này cho phép lưu file ở định dạng PDF tiêu chuẩn, thích hợp để chỉnh sửa và in ấn. - Kích thước tối thiểu: lựa chọn này cho phép lưu file PDF với dung lượng nhỏ nhất, thích hợp cho việc chia sẻ file trực tuyến. Ngoài ra, bạn cũng có thể chỉnh sửa thêm một số tùy chọn nâng cao cho file PDF. Tuy nhiên, bạn nên giữ mặc định những tùy chọn này khi chưa sử dụng thành thạo PDF. Bước 5: Hoàn thành việc chuyển Word sang PDF trên Word 2016, các bạn vào thư mục đã lưu file PDF và kiểm tra kết quả.  Lê Kha
